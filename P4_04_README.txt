Pour faire fonctionner le site à partir de votre hébergeur, vous devrez modifier certains paramètres de la base de données

dans le manager :

Indiquer votre host (le serveur web) (ex : nomserveur.mysql.db),

Votre identifiant,

Votre mot de passe.

N'oubliez pas d'importer la base de données que je vous ai fourni sur le site de votre hébergeur pour récupérer les données du site.
